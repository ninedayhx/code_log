# 图
## 图的数据结构
### 邻接矩阵
图分为有向图和无向图，使用二维数组graph来表示矩阵，其中graph[i][j]表示节点i->j的权重，
所以无向图中graph[i][j]=graph[j][i]，无向图是对称阵

## 常用算法
### DFS
广度优先搜索大多使用如下模板
```cpp
void dfs(){
    if(不符合搜索条件，或者已经访问过)
        return;
    if(符合条件)
        执行一些操作;
    
    dfs(从本节点拓展出的所有搜索条件)
    ···
}
```
注意，深度搜索时需要知道节点是否访问过，
